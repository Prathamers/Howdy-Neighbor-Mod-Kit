Welcome to the **Howdy Neighbor Prototype Mod Kit**. This toolkit provides the essential building blocks for creating stealth-horror levels, advanced AI routines, and physics-based puzzles.

---

## 🛠 Features & Integration
* **Neighbor AI 2.0:** Pre-configured navigation and detection logic.
* **Terrain 3D (Tokisan):** Optimized for large-scale outdoor environments.
* **netGL Library:** High-performance C# utilities for game logic.
* **PSScript Support:** Integrated custom scripting for modders.

---

## 🚀 Quick Start
1. **Clone the Repository:** `git clone https://github.com/ppshe/howdy-neighbor-modkit.git`
2. **Open in Godot:** Import the project into **Godot 4.x**.
3. **Inherit Base Scenes:** * Right-click `Base_Interactable.tscn` -> **"New Inherited Scene"**.
   * Save all custom work in `res://mods/your_name/`.

---

## 📦 How to Export your Mod
To share your level or assets, you must export them as a **Resource Pack**:
1. Go to **Project > Export**.
2. Select your platform (Windows/Linux).
3. Click **"Export PCK/ZIP"**.
4. ⚠️ **NEVER** export as an `.exe` or `.apk`.
